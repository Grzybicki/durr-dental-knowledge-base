# VistaSoft Cloud Exchange — Échange d'images dentaires anonymisées

## Description courte

**VistaSoft Cloud Exchange** est le module Dürr Dental d'**échange d'images
dentaires** entre confrères, laboratoires, cliniques et patients via le cloud.
Le partage se fait en un clic, avec une **anonymisation systématique des
données patient avant transmission** et une **conservation cloud de 30 jours**.

Le module est conforme au **Règlement Général sur la Protection des Données
(RGPD)** européen.

## Identification du module

| Champ | Valeur |
|---|---|
| Nom commercial | VistaSoft Cloud Exchange |
| Catégorie | Plateforme d'échange d'images dentaires |
| Éditeur | Dürr Dental SE — Bietigheim-Bissingen, Allemagne |
| Statut réglementaire | Module utilitaire (non classé dispositif médical) |

## Périmètre fonctionnel

Verbatim de la [page produit officielle VistaSoft Cloud Exchange](https://www.duerrdental.com/en/products/software/imaging/vistasoft-cloud-exchange/) :

> *« VistaSoft Cloud Exchange offers the perfect platform for sharing images
> easily, cost-effectively and efficiently with colleagues, laboratories,
> clinics or patients. With just one click and the corresponding e-mail
> address you can share and comment on images in the cloud. »*

Fonctionnalités clés :

- **Partage en un clic** par email.
- **Commentaires** possibles dans le cloud sur les images partagées.
- **Workflow simplifié** comparé à la copie sur support physique (clé USB,
  CD/DVD).
- **Économies de temps et de coût** par rapport aux méthodes traditionnelles.

## Protection des données et conformité RGPD

Verbatim de la page produit officielle :

> *« The image data is transmitted via the VistaSoft Cloud Exchange in
> accordance with the requirements of the General Data Protection Regulation
> (GDPR), and VistaSoft Cloud Exchange satisfies the requirements of GDPR –
> the strictest data protection regulations in the world. Images are always
> anonymised before being saved to the cloud, as the data that identifies
> the patient is always removed prior to the upload. Afterwards the image
> data is available in the cloud for 30 days. »*

Synthèse des engagements documentés :

| Élément | Valeur |
|---|---|
| Conformité réglementaire | **RGPD (Règlement UE 2016/679)** |
| Anonymisation | **Systématique avant upload cloud** — les données identifiant le patient sont retirées avant transmission |
| Durée de conservation | **30 jours** après upload |
| Géographie des serveurs | À confirmer via Centre de téléchargements |

## Articulation avec VistaSoft

VistaSoft Cloud Exchange s'intègre au logiciel socle
[VistaSoft 4.0](../vistasoft-4-0/overview/). Les images partagées sont
préalablement produites et traitées dans VistaSoft 4.0 (DM IIb), puis
anonymisées et transmises via Cloud Exchange (non-DM).

Pour la **visualisation** des images dans le navigateur (côté destinataire),
le module complémentaire est [VistaSoft Cloud View](../vistasoft-cloud-view/overview/).

## Limites et précisions

- VistaSoft Cloud Exchange est conçu pour le **partage**, pas pour le
  diagnostic clinique primaire.
- L'anonymisation est systématique : les images partagées **ne contiennent
  plus de données identifiant le patient**.
- La conservation cloud est limitée à **30 jours** — les destinataires doivent
  télécharger ou consulter les images dans ce délai.
- Le respect de l'obligation d'information du patient (RGPD) avant tout partage
  reste de la responsabilité du praticien.
