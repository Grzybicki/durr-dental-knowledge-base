# VistaScan Mini View 2.0 — Scanner de plaques au phosphore avec écran

## Description courte

**VistaScan Mini View 2.0** est le **scanner de plaques au phosphore**
compact Dürr Dental qui ajoute au Mini Easy 2.0 un **écran en verre haute
résolution** (ScanManager + informations patient) et le **fonctionnement
autonome** (travail possible sans ordinateur). Il intègre la **technologie
PCS** (Plate Carrier Scanning), la connexion **Wi-Fi** et est livré en
**« SmartScan Edition »** avec **deux VistaScan Smart Reader** inclus pour
le workflow multi-patients.

## Statut réglementaire

Le **VistaScan Mini View 2.0** est, en tant que **scanner** (lecteur de
plaques au phosphore sans fonction de mesure), un **dispositif médical de
classe I** — **auto-déclaration** MDR EU 2017/745, **sans** organisme notifié.
Les **écrans à mémoire VistaScan IQ** utilisés sont, eux, des dispositifs de
classe **IIa** (NB DQS 0297) — voir la
[fiche écrans VistaScan IQ](/durr-dental-knowledge-base/docs/fr/imagerie/vistascan-iq-ecrans/overview/).

## Caractéristiques principales

| Paramètre | Valeur |
|---|---|
| Technologie | **PCS** (Plate Carrier Scanning) |
| Écran | **Écran en verre haute résolution** — ScanManager + informations patient |
| Fonctionnement autonome | **Oui** (travail sans ordinateur, transfert différé des images) |
| Formats de plaques | **S0, S1, S2, S3, S4** (tous les guides de plaque livrés) |
| Plaques compatibles | **VistaScan Image Plates IQ** (RFID + QR) — voir [fiche écrans IQ](/durr-dental-knowledge-base/docs/fr/imagerie/vistascan-iq-ecrans/overview/) |
| Résolution | **40 LP/mm** (théorique) / **22 LP/mm** effective (limitée par l'écran) |
| Connexion | **LAN + Wi-Fi** (2,4 / 5 GHz, WPA / WPA2 / WPA3) |
| Effacement | Automatique post-scan |
| Logiciel | [VistaSoft 4.0](/durr-dental-knowledge-base/docs/fr/imagerie/vistasoft-4-0/overview/) (natif) + Connect/TWAIN |

## Édition SmartScan — 2 Smart Reader inclus

Le **VistaScan Mini View 2.0** est commercialisé en **« SmartScan Edition »** :
il est livré avec **deux [VistaScan Smart Reader](/durr-dental-knowledge-base/docs/fr/imagerie/smart-reader/overview/)**
(lecteurs RFID USB) qui permettent d'**associer chaque plaque IQ à un patient
avant l'acquisition**. Le cabinet peut ainsi **scanner les plaques de plusieurs
patients dans n'importe quel ordre**, les images étant automatiquement rangées
dans le bon dossier. Le Smart Reader est compatible **uniquement** avec le
Mini Easy 2.0 et le Mini View 2.0.

## Contenu de livraison

Le VistaScan Mini View 2.0 est livré avec : bloc secteur, câbles (alimentation,
réseau RJ45, clé USB), **2 VistaScan Smart Reader**, **les guides de plaque T0,
T1, T2, T3 et T4**, **2 écrans à mémoire IQ T0 + 4 écrans à mémoire IQ T2**,
**100 sachets T0 + 300 sachets T2**, rack de rangement des guides, boîte de
rangement des écrans, capot de protection, lingettes nettoyantes, module de
radiographie VistaSoft, VistaSoft Connect/TWAIN et l'outil de service VistaScan.

⚠️ Le kit d'**écrans à mémoire** livré (2× T0 + 4× T2) est identique à celui du
Mini Easy 2.0 ; ce qui distingue le Mini View 2.0, c'est l'**écran embarqué**,
le **Wi-Fi**, le **fonctionnement autonome**, les **2 Smart Reader** et **tous
les guides de plaque** livrés d'origine.

## Questions fréquentes

### Qu'apporte le VistaScan Mini View 2.0 par rapport au Mini Easy 2.0 ?

Un écran en verre (ScanManager + informations patient), le fonctionnement autonome sans ordinateur, la connexion Wi-Fi, deux VistaScan Smart Reader inclus et tous les guides de plaque (T0 à T4) livrés d'origine. Le moteur de scan (PCS, 40/22 LP/mm) et les plaques IQ sont identiques.

### Le VistaScan Mini View 2.0 peut-il fonctionner sans ordinateur ?

Oui : grâce à son écran et à son mode autonome, il peut scanner et enregistrer temporairement les images, transmises ensuite à la base de données.

### Quelles plaques utilise le VistaScan Mini View 2.0 ?

Les plaques VistaScan Image Plates IQ (avec puce RFID et code QR), dans tous les formats intra-oraux S0 à S4.

### Quelle est la classe réglementaire du VistaScan Mini View 2.0 ?

En tant que scanner, il est dispositif médical de classe I (auto-déclaration MDR, sans organisme notifié) ; les écrans à mémoire IQ associés sont de classe IIa.
