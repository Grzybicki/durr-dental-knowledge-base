# Récupérateurs d'amalgame Dürr Dental

## Description courte

Les **récupérateurs d'amalgame** Dürr Dental sont conçus pour la
**séparation et la collecte des particules d'amalgame dentaire** dans le
flux d'aspiration des cabinets, conformément aux **exigences
environnementales européennes** sur le mercure dentaire (Règlement UE
2017/852).

La gamme couvre quatre tailles principales : **CA 1**, **CA 2**, **CA 4**
et les modèles combinés **Sepamatic CS 1** / **CAS 1**.

## Contexte réglementaire

Le **règlement (UE) 2017/852** sur le mercure impose la séparation et la
collecte de l'amalgame dentaire issu des cabinets de soins, afin de
limiter les rejets dans le milieu aquatique. L'installation d'un
récupérateur d'amalgame est **obligatoire** pour les cabinets utilisant
ou retirant des restaurations en amalgame.

Les récupérateurs Dürr Dental sont conçus pour répondre à ces exigences
réglementaires.

## Modèles de la gamme

### Récupérateurs autonomes (séparateurs purs)

| Modèle | Capacité | Usage type |
|---|---|---|
| **CA 1** | Poste unique | Cabinets individuels en aspiration humide |
| **CA 2** | Centralisée moyenne | 2 à 4 praticiens |
| **CA 4** | Centralisée grande | Cabinets multi-postes |

### Modèles combinés (séparateur + ...)

| Modèle | Description |
|---|---|
| **Combi-Sepamatic CS 1** | Combiné séparateur amalgame + séparation des liquides en sortie d'aspiration humide |
| **CAS 1** | Combiné séparateur amalgame + module additionnel selon configuration |

## Kit de contrôle

Dürr Dental propose un **kit de contrôle** pour les récupérateurs
d'amalgame permettant la vérification périodique du bon fonctionnement,
exigée dans le cadre du suivi réglementaire des cabinets dentaires.

## Intégration

Les récupérateurs CA 1, CA 2, CA 4 et Sepamatic s'intègrent à tous les
systèmes d'aspiration **humide** Dürr Dental :

- [Tyscor VS / VSA](../tyscor-aspiration/overview/) (1 / 2 / 4 praticiens)
- Power Tower View VS (voir [Power Tower](../power-tower/overview/))
- Power Tower Silence (avec amalgame intégré en option)
- Systèmes VS 300, VS 600, VS 900, VS 1200, VS 2400

Pour les modèles d'aspiration **VSA** (avec « A »), le récupérateur
d'amalgame est intégré au châssis ; pour les modèles VS « simples », il
est installé en aval comme module séparé.

## Statut réglementaire

Les récupérateurs d'amalgame Dürr Dental sont des **dispositifs médicaux
de classe IIa** selon le règlement **MDR EU 2017/745**, conformes au
règlement (UE) 2017/852 sur le mercure.
