# Tornado — Compresseurs dentaires Dürr Dental

## Description courte

**Tornado** est la nouvelle génération de compresseurs dentaires Dürr
Dental, conçue pour combiner **performance**, **silence** et
**efficacité énergétique**. La gamme couvre trois modèles : **Tornado 1**,
**Tornado 2** et **Tornado 4** pour s'adapter à la taille du cabinet.

## Distinctions récentes

| Distinction | Année | Catégorie |
|---|---|---|
| **German Design Award 2026** | 2026 | Excellence en design produit médical |
| **Innovations-Champion Award 2025** — 1ʳᵉ place | 2025 | Innovation produit |

## Statut réglementaire

Les compresseurs Tornado sont des **dispositifs médicaux de classe IIa**
selon le règlement européen **MDR EU 2017/745**. Ils sont conformes aux
exigences strictes de la norme internationale **ISO 22052** qui définit
les critères de qualité de l'air comprimé dentaire.

## Caractéristiques clés

D'après la page produit officielle Dürr Dental :

| Caractéristique | Valeur |
|---|---|
| **Pression maximale** | **8 bar** |
| **Économies d'énergie** | **Jusqu'à 12 %** d'économies par rapport à la génération précédente |
| **Norme qualité d'air** | Conforme **ISO 22052** |
| **Niveau sonore** | Très silencieux (déclinaison **Super Silent** disponible) |
| **Durabilité** | « Built for the long run » — conception orientée durée de vie |

## Modèles de la gamme

| Modèle | Capacité | Usage type |
|---|---|---|
| **Tornado 1** | 1 praticien | Cabinets individuels, postes uniques. Déclinaison **Tornado 1 Super Silent** disponible. |
| **Tornado 2** | 2 praticiens | Cabinets de taille moyenne |
| **Tornado 4** | 4 praticiens | Cabinets de grande taille / multi-praticiens |

## Bénéfices

Selon la documentation officielle Dürr Dental :

- **Performance fiable** sur la durée — *« Strong, reliable performance »*.
- **Économies d'énergie** quotidiennes — *« providing more power where it
  matters while consuming less energy every day »*.
- **Silence de fonctionnement** — la déclinaison Super Silent renforce
  cet aspect pour les cabinets très exigeants.
- **Long cycle de vie** — *« Built for the long run »*.
- **Qualité d'air dentaire conforme ISO 22052**.
