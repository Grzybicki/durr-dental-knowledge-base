# Hygowater — Traitement de l'eau des fauteuils dentaires

## Description courte

**Hygowater** est l'appareil Dürr Dental de **décontamination de l'eau
des fauteuils dentaires**. Il **élimine les micro-organismes** présents
dans l'eau d'alimentation des unités de soins et offre une **protection
durable contre le biofilm** dans les canalisations.

Le système fonctionne par **filtration et électrolyse**, **sans
utilisation de produits chimiques**.

## Caractéristiques clés

| Paramètre | Valeur |
|---|---|
| Principe de désinfection | **Filtration + électrolyse** |
| Produits chimiques | **Aucun** |
| Niveau de décontamination cible | **< 100 UFC/ml** (Unités Formant Colonies par millilitre) |
| Validation académique | **Faculté dentaire de Strasbourg** (validation publiée) |

Le niveau cible **< 100 UFC/ml** est conforme aux exigences de qualité
microbiologique de l'eau pour soins dentaires telles que recommandées par
les autorités sanitaires européennes et les sociétés scientifiques
odontologiques.

## Bénéfices documentés

D'après les communiqués officiels et les articles presse spécialisée :

- **Élimination des micro-organismes** de l'eau du fauteuil.
- **Protection durable** contre le biofilm dans les canalisations
  d'alimentation.
- **Risque infectieux minimisé** pour le patient et le praticien.
- **Facilité d'utilisation** : installation et exploitation simples.

## Validation académique

Le système Hygowater a fait l'objet d'une **validation par la Faculté
dentaire de Strasbourg**, publiée dans la presse dentaire spécialisée
(Dynamique Dentaire). Cette validation tierce indépendante renforce
l'évidence d'efficacité dans des conditions réelles de cabinet.

## Différenciation avec Hygopure

- **Hygopure 90** = traitement d'eau **DI (déminéralisée)** pour
  alimenter les **stérilisateurs Hygoclave** (voir
  [fiche Hygoclave + Hygopure](../hygoclave-hygopure/overview/)).
- **Hygowater** = décontamination de l'eau des **fauteuils dentaires**
  pendant les soins (eau de spray, eau de turbines).

Les deux produits sont complémentaires dans la chaîne d'hygiène du
cabinet.
