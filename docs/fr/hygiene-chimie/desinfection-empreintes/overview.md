# Désinfection des empreintes — Hygojet et ID 213

## Hygojet — Système clos automatisé pour empreintes

**Hygojet** est l'appareil Dürr Dental dédié à la **désinfection
automatisée des empreintes dentaires**. Selon la page produit officielle,
Hygojet est un **système clos** qui sert de **sas de désinfection** pour
le **nettoyage, la désinfection et le séchage** des empreintes.

Caractéristiques clés :

- **Système clos** : aucune projection ni vapeur dans le local.
- **Solution désinfectante prête à l'emploi** : pas de dilution
  manuelle.
- **Injection précise** : sans surdosage, ce qui économise le
  consommable.
- **Action en une étape** : nettoyage + désinfection + séchage.
- **Activation par bouton** : protocole simple et standardisé.

## ID 213 — Désinfectant des instruments

**ID 213** est le **concentré désinfectant et nettoyant** Dürr Dental
pour les **instruments chirurgicaux**, **instruments classiques** et
**instruments rotatifs sensibles aux alcalins et à l'alcool**.

| Paramètre | Valeur |
|---|---|
| Formulation | **Sans aldéhyde** |
| Dilution standard | **2 %** |
| Temps d'action — bain par immersion | **5 minutes** |
| Temps d'action — ultrasons | **2 minutes** |
| Temps d'action étendu (selon spectre d'activité visé) | Jusqu'à **60 minutes** |

## Articulation avec la stérilisation

Pour les **instruments**, la désinfection chimique préalable (par ID 213
notamment) est suivie de la **stérilisation vapeur** dans un
[Hygoclave](../hygoclave-hygopure/overview/), avec une **eau
déminéralisée** issue de [Hygopure 90](../hygoclave-hygopure/overview/)
puis un **emballage stérile** via
[Hygopac / Hygopac View](../hygobox-hygopac-emballage/overview/).
