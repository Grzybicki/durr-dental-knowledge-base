# Vector Paro — Thérapie parodontale ultrasonique atraumatique

## Description courte

**Vector Paro** est le système Dürr Dental de **thérapie parodontale
ultrasonique atraumatique**. Il s'appuie sur la **technologie Vector**
propriétaire, développée par Dürr Dental depuis **plus de 25 ans**, qui
permet un traitement **minimalement invasif et indolore** des
parodontites et péri-implantites.

## Principe technique unique — Vibration deflection

Selon la page produit officielle, le principe Vector :

> *« Over 25 years ago, DÜRR DENTAL developed its unique vibration
> deflection and set new standards in the field of painless, minimally
> invasive periodontitis treatment with the Vector System. »*

Le mécanisme physique :

- **Anneau de résonance Vector** propriétaire.
- Les oscillations ultrasoniques sont **redirigées de 90°** par cet
  anneau.
- Au lieu d'un mouvement circulaire ou elliptique classique, le mouvement
  devient **linéaire de haut en bas**.
- Les instruments **se déplacent parallèlement à la surface
  radiculaire**, ce qui permet :
  - Une **action sans vibration latérale** sur le tissu (atraumatique).
  - Un **traitement minimalement invasif** des poches parodontales.

Ce principe se distingue des ultrasons piézoélectriques ou magnétostrictifs
classiques par cette **redirection mécanique linéaire**.

## Action biologique — Vector Fluid

Vector Paro utilise un **fluide de polissage Vector** (Vector Fluid) qui
amplifie l'effet du traitement :

- Élimination du **biofilm bactérien** par action mécanique douce.
- **Polissage** de la surface radiculaire.
- **Action atraumatique** combinée — douleur réduite, préservation des
  tissus mous.

## Instruments et accessoires

Le système Vector Paro est associé à une **large gamme d'instruments**
spécialisés pour les différents types de traitement, disponibles dans
la **boutique en ligne officielle Dürr Dental** :

### Pour thérapies de péri-implantites

Inserts spécifiques conçus pour le débridement des surfaces
implantaires sans altérer l'implant.

### Pour thérapies de parodontites

| Famille d'instruments | Notes |
|---|---|
| **Paro-Sonde gerade** (droite) | Sonde parodontale standard |
| **Paro-Sonde gebogen** (courbée) | Pour zones moins accessibles |
| **Paro-Sonde Plus** | Variante améliorée |
| **Paro-Lanzette** | Lancette parodontale |
| **Paro-Kürette** | Curette parodontale |
| **Recall-Kürette CFK** | Curette pour suivi de maintenance |

### Vector Scaler

Le **Vector Scaler** est le module détartrage du système — couvre les
besoins de détartrage classique en complément de la thérapie parodontale
Vector Paro.

### Vector Consommables

Vector Fluid de polissage + accessoires de maintenance.

## Indications cliniques

| Indication | Traitement Vector |
|---|---|
| **Parodontite** (légère à sévère) | Vector Paro avec inserts parodontite |
| **Péri-implantite** | Vector Paro avec inserts péri-implantites |
| **Maintenance parodontale** (recall) | Vector Paro avec Recall-Kürette CFK |
| **Détartrage** | Vector Scaler |

## Articulation avec Lunos

Vector Paro et [Lunos](../lunos-system/overview/) sont les deux systèmes
Dürr Dental complémentaires pour la chaîne préventive et thérapeutique
parodontale :

- **Lunos** = prophylaxie générale, aéropolissage, pâtes, fluor.
- **Vector** = thérapie parodontale ultrasonique atraumatique
  (parodontites, péri-implantites).
