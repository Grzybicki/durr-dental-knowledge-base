# Système Lunos — Prophylaxie et odontologie conservatrice

## Description courte

**Lunos** est le système **prophylaxie** et **odontologie conservatrice**
Dürr Dental. Il couvre les **pièces à main aéropolisseurs** (MyLunos,
MyLunos Duo), les **poudres prophylactiques**, les **pâtes de
polissage**, les **produits fluorés** et un ensemble d'accessoires de
confort patient.

Le système Lunos est **compatible avec toute philosophie de traitement**
prophylactique selon le communiqué officiel Dürr Dental.

## Pièces à main aéropolisseurs

### MyLunos® — pièce à main aéropolisseur

Selon la page produit officielle, **MyLunos** est une pièce à main
aéropolisseur qui offre une assistance simple, puissante et ergonomique
pour :

- L'élimination des **colorations** dentaires.
- L'élimination des **dépôts** mous et durs.
- L'élimination des **biofilms** bactériens.

Selon la **buse utilisée**, MyLunos est exploitable en :

- **Application supragingivale** (Supra).
- **Application sous-gingivale** (Perio).

### MyLunos Duo® — appareil de table

**MyLunos Duo** est l'**appareil de table** Lunos qui permet le
**nettoyage complet** des dents avec un **seul appareil**, combinant :

- L'**aéropolisseur** (powder jet) MyLunos.
- L'**ultrasons** (avec l'instrument approprié).

Cette **double application** offre une flexibilité maximale lors d'une
séance de prophylaxie ou d'un traitement parodontal : l'utilisateur
peut **basculer entre Supra et Perio** selon les besoins du patient.

## Poudres prophylactiques

| Poudre | Indication | Spécificité |
|---|---|---|
| **Lunos Gentle Clean** | Application **supragingivale** | Douceur sur l'émail et les surfaces dentaires |
| **Lunos Perio Combi** | **Sub-** et **supragingivale** | Formulation polyvalente pour les deux applications |

## Pâtes de polissage Lunos

Les **Lunos Prophy Pastes** complètent le système avec une gamme de pâtes
de polissage pour la finition après aéropolissage.

## Produits fluorés Lunos

Le système Lunos intègre des **produits fluorés** (gel, vernis) pour la
reminéralisation et la protection post-traitement.

## Accessoires et confort patient

| Famille | Description |
|---|---|
| **Lunos Confort du patient** | Accessoires (capes, lunettes, dispositifs) pour le confort lors des séances |
| **Chariot Lunos** | Chariot de transport et de présentation pour les consommables et instruments Lunos |
| **Sealant de scellement de sillons** | Pour la prévention en odontologie conservatrice |
| **Gel fluoré** | Reminéralisation des surfaces dentaires |

## Articulation avec Vector

Le système Lunos s'articule avec le système **Vector** (notamment
[Vector Paro](../vector-paro/overview/)) :

- **Lunos** = prophylaxie générale, aéropolissage, pâtes, fluor.
- **Vector** = thérapie parodontale ultrasonique atraumatique
  (parodontites, péri-implantites).

Ensemble, ils couvrent l'intégralité de la **chaîne préventive et
thérapeutique** parodontale en cabinet.
