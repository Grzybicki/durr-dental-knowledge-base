# Hygoclave et Hygopure — Stérilisation et traitement de l'eau

## Description courte

La gamme **Hygoclave** de Dürr Dental couvre les **stérilisateurs vapeur**
classe B pour cabinets dentaires : **Hygoclave 40**, **Hygoclave 50** et
**Hygoclave 90**. Les systèmes **Hygopure 90** et **Hygodem 90** assurent
le **traitement de l'eau déminéralisée** nécessaire à l'alimentation
des stérilisateurs.

## Hygoclave — Stérilisateurs vapeur

### Hygoclave 40

Selon la page produit officielle, **Hygoclave 40** est un stérilisateur
**classe B** répondant aux **plus hautes exigences professionnelles**
en cycles de stérilisation. Il représente une **solution simple et
fiable** pour les cabinets dentaires de taille standard.

### Hygoclave 50

**Hygoclave 50** est la solution **professionnelle** pour les cabinets
exigeants. Caractéristiques :

- Cycles de stérilisation **classe B**.
- Capacité adaptée aux cabinets de taille intermédiaire.
- Intégration possible avec le système de traitement d'eau Hygopure 90 /
  Hygodem 90.

### Hygoclave 90

**Hygoclave 90** complète la **chaîne d'hygiène système** Dürr Dental
avec une technologie de pointe. Caractéristiques techniques officielles :

| Paramètre | Valeur |
|---|---|
| **Pompe vacuum** | **3 pistons** |
| **Puissance de chauffe** | **2400 W** |
| **Matériau composants en contact** | **V4A en acier inoxydable** |
| **Fiabilité** | Composants premium, cycles rapides |
| **Efficacité énergétique** | Très élevée |

L'utilisation exclusive de l'**acier inoxydable V4A** sur tous les
composants pertinents protège de manière optimale les instruments et
empêche le **transfert de matériau** lors de la stérilisation vapeur.

## Hygopure et Hygodem — Traitement de l'eau

Les stérilisateurs vapeur exigent une **eau déminéralisée (DI)** de
qualité pour fonctionner correctement et éviter l'**entartrage**.

### Hygopure 90

Selon la page produit officielle, **Hygopure 90** permet l'**extraction
de grandes quantités d'eau DI** (plusieurs litres) en une seule fois.
Caractéristiques :

- **Durée de vie élevée** des cartouches et de la membrane.
- **Fiabilité** à haute fréquence d'utilisation.
- Conçu pour les **grands cabinets** et **cliniques**.

### Hygodem 90

**Hygodem 90** est le **système complémentaire** de traitement d'eau
Dürr Dental, conçu pour les besoins plus modestes ou les configurations
spécifiques de cabinet.

## Statut réglementaire

Les stérilisateurs Hygoclave sont des **dispositifs médicaux**
réglementés selon **MDR EU 2017/745**. Ils doivent respecter la norme
**EN 13060** pour les petits stérilisateurs vapeur (classes N, S et B).

La **classe B** désigne les stérilisateurs capables de traiter tous
types de charges, y compris :

- Instruments creux (turbines, contre-angles).
- Charges emballées en sachets stériles.
- Charges poreuses (textiles, cotons).

## Chaîne d'hygiène système Dürr Dental

L'écosystème Hygoclave + Hygopure s'inscrit dans la **chaîne d'hygiène
système** complète Dürr Dental, qui couvre :

1. **Préparation** : Hygoclave (stérilisation vapeur).
2. **Traitement d'eau** : Hygopure 90 / Hygodem 90.
3. **Emballage stérile** : Hygopac / Hygopac View (voir
   [fiche Hygobox / Hygopac](../hygobox-hygopac-emballage/overview/)).
4. **Documentation** : Hygoprint pour la traçabilité.
5. **Stockage** : Hygobox.
6. **Désinfection préalable** : produits chimiques (voir fiches
   désinfection [aspiration](../desinfection-aspiration/overview/),
   [surfaces](../desinfection-surfaces/overview/),
   [empreintes](../desinfection-empreintes/overview/)).
