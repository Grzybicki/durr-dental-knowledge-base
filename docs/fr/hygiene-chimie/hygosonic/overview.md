# Hygosonic — Bac à ultrasons pour préparation des instruments

## Description courte

**Hygosonic** est le **bac à ultrasons** Dürr Dental dédié au
**nettoyage rapide et intense** des instruments dentaires en amont de la
stérilisation. Il s'inscrit dans la **chaîne de préparation manuelle**
des instruments du cabinet.

## Caractéristiques clés

D'après la page produit officielle :

- **Désinfection et nettoyage** efficaces sans agressivité pour les
  instruments.
- **Champ sonore optimisé** pour un nettoyage particulièrement
  homogène — pas de zones mortes.
- **Fonction « Boost »** : augmentation de la puissance pour les
  charges particulièrement encrassées.
- **Fonction « Dégazage »** : purge de l'air des solutions, condition
  pour une efficacité ultrasonique maximale.

## Combinaison avec ID 213 et autres produits ID/MD

Hygosonic est conçu pour être utilisé avec les produits chimiques
désinfectants Dürr Dental de la série **ID** (Instrument Disinfection)
et **MD** (Material Disinfection) :

- [ID 213](../desinfection-empreintes/overview/) : concentré
  désinfectant et nettoyant aldéhyde-free pour instruments chirurgicaux,
  classiques et rotatifs.

L'utilisation combinée garantit la **désinfection sécurisée** et le
**nettoyage en une seule étape**.

## Place dans la chaîne d'hygiène

1. **Préparation manuelle** : Hygosonic (ultrasons + ID 213) ← cette
   étape.
2. **Transport** : [Hygobox](../hygobox-hygopac-emballage/overview/).
3. **Stérilisation vapeur** :
   [Hygoclave 40 / 50 / 90](../hygoclave-hygopure/overview/) avec eau DI
   Hygopure 90.
4. **Emballage stérile** :
   [Hygopac / Hygopac View](../hygobox-hygopac-emballage/overview/) +
   Hygofol.
5. **Documentation** : Hygoprint.
